## Descripción del Proyecto

El proyecto busca ser una plataforma interactiva, en el que las personas puedan observar el funcionamiento de los algoritmos de encriptación, de forma que al conocer el funcionamiento se disminuya la ofuscación existente por el desconocimiento del uso de estos algoritmos. Además de resaltar la importancia de las matemáticas discretas dentro de algunos de los algoritmos de encriptación más importantes en la actualidad.
